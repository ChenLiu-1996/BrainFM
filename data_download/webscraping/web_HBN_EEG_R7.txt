            <h2>
                <a class="headerlink" href="#aws-and-cyberduck"></a>
                AWS and Cyberduck
            </h2>
            <p>
                MRI and EEG data, organized into folders by participant, may also be accessed through an Amazon Web Services (AWS) <a class="reference external" href="https://aws.amazon.com/s3/">S3 bucket</a>
                .
            </p>
            <p>
                Each file in the S3 bucket can only be accessed using HTTP (i.e., <strong>no ftp or scp</strong>
                ). You can obtain a URL for each desired file and then download it using an HTTP client such as a web browser, <em>wget</em>
                , or <em>curl</em>
                . Each file can only be accessed using its literal name- wildcards will not work.
            </p>
            <p>
                There are file transfer programs that can handle S3 natively and will allow you to navigate through the data using a file browser.
                <a class="reference external" href="https://cyberduck.io/">
                    <b>Cyberduck</b>
                </a>
                is one such program that works with Windows and Mac OS X. Cyberduck also has a <a class="reference external" href="https://duck.sh">command line version</a>
                that works with Windows, Mac OS X, and Linux. Instructions for using Cyberduck are as follows:
            </p>
            <ol class="arabic simple">
                <li>
                    Open Cyberduck and click on <em>Open Connection</em>
                    .
                </li>
                <li>
                    Set the application protocol in the dropdown menu to <em>S3 (Amazon Simple Storage Service)</em>
                    .
                </li>
                <li>
                    Set the server to <em>s3.amazonaws.com</em>
                    .
                </li>
                <li>
                    Check the box labelled <em>Anonymous Login</em>
                    .
                </li>
                <li>
                    Expand the More Options tab

                    <ul>
                        <li>
                            To access compressed files (.tar.gz), set Path to <em>fcp-indi/data/Archives/HBN</em>
                        </li>
                        <li>
                            To access uncompressed files, set Path to <em>fcp-indi/data/Projects/HBN</em>
                        </li>
                    </ul>
                <li>
                    Click <em>Connect</em>
                    .
                </li>
            </ol>
